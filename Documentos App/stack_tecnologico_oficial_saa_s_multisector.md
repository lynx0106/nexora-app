# Stack Tecnológico Oficial – SaaS Multisector

## Principios
- Bajo costo inicial
- Escalabilidad horizontal
- Seguridad enterprise
- Modularidad

---

## Frontend
- **Next.js (React)**
- Tailwind CSS
- Zustand / React Query
- Autenticación JWT

Funciones:
- Dashboard negocio
- Dashboard superadmin
- Catálogos
- Pedidos
- Perfil usuario con foto/avatar

---

## Backend
- **NestJS (Node.js)**
- Arquitectura hexagonal
- API REST + Webhooks
- RBAC (roles)

Módulos:
- Auth
- Usuarios
- Negocios
- Pedidos
- Catálogo
- IA
- Métricas

---

## Base de Datos
- PostgreSQL
- Esquema multi-tenant
- Redis (cache y sesiones)

## IA (Pluggable)
