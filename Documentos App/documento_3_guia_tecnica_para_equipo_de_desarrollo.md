# Guía Técnica – Ejecución del Producto

## Objetivo
Servir como guía clara para el equipo técnico durante la ejecución.

---

## Stack
- Frontend: React + Next.js
- Backend: NestJS
- DB: PostgreSQL
- Cache: Redis
- Infra: Cloud + Docker

---

- IA desacoplada

- Logs obligatorios
